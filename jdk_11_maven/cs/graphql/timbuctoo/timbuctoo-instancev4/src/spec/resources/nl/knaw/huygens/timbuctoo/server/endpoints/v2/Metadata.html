The metadata API allows a client to introspect the data model that is configured for a specific VRE. This allows you to build generic clients that work for any VRE without knowledge of the specific configuration at build time.

Retrieving the configuration for a VRE

When a GET is performed to metadata/vre

GET /v2.1/metadata/WomenWriters HTTP/1.1 Content-type: application/json

The server will show a list of metadata about the data fields that can be used to generate an edit or search interface
