The autocomplete endpoint allows you to search through a collection using substrings of the displayname.

Retrieving some results

When a GET is performed to {collection}/autocomplete

GET /v2.1/domain/wwlanguages/autocomplete?query=*d* HTTP/1.1 Content-type: application/json

The server will show all languages that contain a "d"
